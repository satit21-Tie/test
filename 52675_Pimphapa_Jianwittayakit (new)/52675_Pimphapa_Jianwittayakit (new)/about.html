Description (from PDF)

Fish–Eel–Stingray (FES) Classifier is a web application that performs on‑device image classification to categorize an uploaded aquatic image into three classes: Fish, Eel, or Stingray. The model was trained using Google Teachable Machine (TensorFlow.js export). The dataset was curated for learning purposes from public web imagery and free stock photography.

AI Type

Image Classification using convolutional neural networks with depthwise separable layers (MobileNet‑style backbone) and a dense softmax head.

Data Sources

  • Google Images — for general examples (educational use)
  • Unsplash — free photos
  • Pixabay — free photos

How to Use

  1. Open Try the Model page.
  2. Upload a clear image of a fish, an eel, or a stingray.
  3. Read the prediction and confidence score in real time.
